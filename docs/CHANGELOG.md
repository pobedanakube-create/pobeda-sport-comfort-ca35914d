# Журнал изменений (Changelog)

Все значимые изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
проект придерживается [Семантического версионирования](https://semver.org/lang/ru/).

---

## [1.1.0] — 2025-01-25

### Добавлено
- Интеграция с YCLIENTS для онлайн-записи
- Компонент `YClientsBooking` с iframe виджетом
- Система версионности (`src/lib/version.ts`)
- Отображение версии в footer сайта
- Компонент `PerformanceMonitor` для отслеживания метрик
- Компонент `SEOHead` для мета-тегов
- Компонент `LazyImage` для оптимизированной загрузки изображений

### Улучшено
- Оптимизирована адаптивность на мобильных устройствах
- Улучшена производительность загрузки страницы
- Добавлены анимации fade-in для секций

### Исправлено
- Корректное отображение на экранах < 375px

---

## [1.0.0] — 2025-01-24

### Добавлено
- Первоначальный релиз сайта
- Компонент `Header` с адаптивной навигацией
- Компонент `HeroBanner` с фоновым изображением
- Компонент `AboutSection` с описанием клуба
- Компонент `ServicesSection` со списком услуг
- Компонент `Gallery` с фотогалереей
- Компонент `Testimonials` с отзывами клиентов
- Компонент `Benefits` с преимуществами
- Компонент `BookingForm` для записи на тренировку
- Компонент `Features` с дополнительными возможностями
- Компонент `ContactSection` с картой Mapbox
- Полный набор UI-компонентов shadcn/ui
- Адаптивный дизайн (mobile-first)
- Tailwind CSS конфигурация с кастомными цветами

### Технологии
- React 18.3.1
- Vite 5.4.19
- TypeScript 5.8.3
- Tailwind CSS 3.4.17
- shadcn/ui (Radix UI)
- React Router DOM 6.30.1
- Mapbox GL 3.15.0

## Типы изменений

- **Добавлено** — новый функционал
- **Изменено** — изменения в существующем функционале
- **Устарело** — функционал, который будет удалён
