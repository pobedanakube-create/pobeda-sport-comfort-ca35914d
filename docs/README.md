# СК Победа — Официальный сайт спортивного клуба

## Общая информация

**Проект:** Сайт-визитка спортивного клуба «Победа»  
**Версия:** 1.3.1  
**Дата создания:** Январь 2025  
**Последнее обновление:** Декабрь 2025  
**Статус:** Production  

## Цель проекта

Создание современного, адаптивного веб-сайта для спортивного клуба «СК Победа» в г. Калуга с целью:
- Привлечения новых клиентов
- Онлайн-записи на тренировки через форму с Telegram-уведомлениями
- Информирования о услугах и ценах
- Отображения контактной информации и местоположения
- Соответствие законодательству РФ (152-ФЗ о персональных данных)

## Основные функции

1. **Главный баннер (Hero)** — привлекающий внимание блок с CTA-кнопками
2. **О клубе** — описание преимуществ и особенностей
3. **Услуги** — список предоставляемых услуг с ценами
4. **Галерея** — фотографии клуба (WebP для оптимизации)
5. **Отзывы** — реальные отзывы клиентов с Яндекс Карт
6. **Форма записи** — с отправкой заявок в Telegram
7. **Контакты** — карта Яндекс, контактные данные
8. **Юридические документы** — Политика конфиденциальности и Пользовательское соглашение (PDF)
9. **Cookie Consent** — баннер согласия на обработку ПД (152-ФЗ)

## Технологический стек

- **Frontend Framework:** React 18.3
- **Build Tool:** Vite 5.4
- **Language:** TypeScript 5.8
- **Styling:** Tailwind CSS 3.4
- **UI Components:** shadcn/ui (Radix UI)
- **State Management:** React Query (TanStack)
- **Routing:** React Router DOM 6.30
- **Maps:** Яндекс Карты (iframe)
- **Forms:** React Hook Form + Zod
- **Icons:** Lucide React
- **Notifications:** Telegram Bot API

## Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/pobedanakube-create/pobeda-sport-comfort-ca35914d.git

# Переход в директорию
cd pobeda-sport-comfort-ca35914d

# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build
```

## Telegram Bot для заявок

Заявки с формы записи отправляются в Telegram. Настройка:

1. Создайте бота через @BotFather
2. Получите токен бота и chat_id
3. Добавьте в конфигурацию формы
